# Welcome to Mi Compilador

Mi Compilador is a compiler and interpreter for a custom programming language that demonstrates fundamental compiler design principles using ANTLR4. It supports integer variables, arithmetic operations, relational comparisons, and if-else control flow.

## Example Program

Here's a simple program in Mi Compilador's language:

```text Example.txt
program {
    int x;
    int y;
    int resultado;
    
    x = 10;
    y = 20;
    
    // Arithmetic with precedence
    resultado = (x + y) * 2 / 5;
    
    // Conditional logic
    if (resultado > 5) {
        x = 1;
    } else {
        x = 0;
    }
}
```

This program:
1. Declares three integer variables
2. Performs arithmetic calculations with proper operator precedence
3. Uses conditional logic based on relational comparisons
4. Manages variable values in memory throughout execution

## How It Works

Mi Compilador processes source code through several stages:

1. **Lexical Analysis** - The lexer tokenizes the source code into meaningful tokens
2. **Parsing** - The parser builds an Abstract Syntax Tree (AST) from tokens
3. **Execution** - The visitor pattern traverses the AST and executes instructions
4. **Memory Management** - Variables are stored and retrieved from runtime memory

<Note>
Mi Compilador is designed as an educational tool to demonstrate compiler construction principles. It showcases how ANTLR4 can be used to build a complete language processor with minimal code.
</Note>
