Get your first program running with Mi Compilador and learn the basics of the language.

## Prerequisites

Before you begin, ensure you have:
- Python 3.x installed
- ANTLR4 runtime for Python (`pip install antlr4-python3-runtime`)

## Run Your First Program

<Steps>
  <Step title="Navigate to the compiler directory">
    Open your terminal and navigate to where Mi Compilador is located:
    
    ```bash
    cd path/to/mi-compilador
    ```
  </Step>

  <Step title="Run the example program">
    Mi Compilador comes with example programs. Let's run the first one:
    
    ```bash
    python3 Prueba.py Entrada1.txt
    ```
    
    You should see output like this:
    
    ```
    [OK] Archivo 'Entrada1.txt' cargado correctamente.
    ------------------------------
    Asignación: x = 10
    Asignación: y = 20
    Asignación: resultado = 12
    Asignación: control = 1
    Asignación: resultado = 22
    ------------------------------
    Ejecución finalizada con éxito.
    Estado de la memoria: {'x': 10, 'y': 20, 'resultado': 22, 'control': 1}
    ```
  </Step>

  <Step title="Understand what happened">
    The compiler processed `Entrada1.txt` through three stages:
    
    1. **Lexical Analysis**: Tokenized the source code
    2. **Syntax Parsing**: Built an Abstract Syntax Tree (AST)
    3. **Execution**: Used the Visitor pattern to evaluate expressions and execute statements
    
    The program:
    - Declared variables (`x`, `y`, `resultado`, `control`)
    - Performed arithmetic: `(10 + 20) * 2 / 5 = 12`
    - Evaluated a conditional: Since `12 > 5`, it set `control = 1`
    - Updated the result: `12 + 10 = 22`
  </Step>

  <Step title="Try the second example">
    Run another example to see conditional logic in action:
    
    ```bash
    python3 Prueba.py Entrada2.txt
    ```
    
    Expected output:
    
    ```
    [OK] Archivo 'Entrada2.txt' cargado correctamente.
    ------------------------------
    Asignación: valorUno = 5
    Asignación: valorDos = 10
    Asignación: resultado = 15
    Asignación: exito = 1
    ------------------------------
    Ejecución finalizada con éxito.
    Estado de la memoria: {'valorUno': 5, 'valorDos': 10, 'resultado': 15, 'exito': 1}
    ```
    
    This program adds two numbers and uses an `if` statement to check if the result exceeds 10.
  </Step>
</Steps>

## Create Your Own Program

Now let's write a simple custom program from scratch.

<Steps>
  <Step title="Create a new file">
    Create a file named `MiPrograma.txt`:
    
    ```bash
    touch MiPrograma.txt
    ```
  </Step>

  <Step title="Write your program">
    Open `MiPrograma.txt` and add this code:
    
    ```
    program {
        int a;
        int b;
        int suma;
        int mayor;
        
        a = 15;
        b = 25;
        suma = a + b;
        
        if (suma > 30) {
            mayor = 1;
        } else {
            mayor = 0;
        }
    }
    ```
    
    This program:
    - Declares four integer variables
    - Assigns values and calculates their sum
    - Checks if the sum is greater than 30
  </Step>

  <Step title="Compile and run">
    Execute your program:
    
    ```bash
    python3 Prueba.py MiPrograma.txt
    ```
    
    You should see:
    
    ```
    [OK] Archivo 'MiPrograma.txt' cargado correctamente.
    ------------------------------
    Asignación: a = 15
    Asignación: b = 25
    Asignación: suma = 40
    Asignación: mayor = 1
    ------------------------------
    Ejecución finalizada con éxito.
    Estado de la memoria: {'a': 15, 'b': 25, 'suma': 40, 'mayor': 1}
    ```
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Error: python3 command not found">
    Make sure Python 3 is installed. Try:
    ```bash
    python --version
    ```
    If it shows Python 3.x, use `python` instead of `python3`.
  </Accordion>

  <Accordion title="Error: No module named 'antlr4'">
    Install the ANTLR4 Python runtime:
    ```bash
    pip install antlr4-python3-runtime
    ```
  </Accordion>

  <Accordion title="Error: El archivo tiene errores gramaticales">
    Check your program syntax:
    - All statements must end with semicolons
    - Variables must be declared before use
    - The program must be wrapped in `program { ... }`
  </Accordion>

  <Accordion title="Error: Variable no definida">
    Ensure you declare all variables with `int` before using them:
    ```
    int myVar;
    myVar = 10;
    ```
  </Accordion>
</AccordionGroup>
